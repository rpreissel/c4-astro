## Browser Popup

### Enabled (default)

Click on the diagram to open the browser popup:

```astro
<LikeC4View viewId="index" browser={true} />
```

<LikeC4View viewId="index" browser={true} />

### Disabled

No popup on click:

```astro
<LikeC4View viewId="index" browser={false} />
```

<LikeC4View viewId="index" browser={false} />

### Custom Browser Configuration

```astro
<LikeC4View 
  viewId="index" 
  browser={{
    background: 'dots',
    controls: true,
    showNavigationButtons: true,
    enableSearch: true,
    enableFocusMode: true,
    enableElementDetails: true,
    enableRelationshipDetails: true,
    enableNotations: true,
  }}
/>
```
